Privacy Policy

Last Updated: January 17, 2026

This Privacy Policy describes how your information is handled when you interact with the abstract project repository and its hosted tutorial website. Your privacy is a priority, and this project is designed with a "Privacy by Design" approach to ensure a secure and anonymous educational experience.

1. Data Collection

We believe in data minimization. As such, this project adheres to the following principles:

  • No Personal Data: This website does not require user registration, accounts, or the input of personal details such as names, email addresses, or phone numbers.
  • No Cookies: We do not use tracking cookies, advertising identifiers, or any persistent tracking technologies to monitor your behavior across the web.
  • Local Processing: Any interactivity on this site, including code highlighting and the search filter in the "Side-by-Side Reference" table, is processed entirely locally on your device via client-side JavaScript. No data is transmitted to our servers or any third-party databases.

2. Third-Party Services

While we do not collect data directly, this project is hosted and delivered using the following standard industry infrastructure:

  • GitHub Pages: If you are viewing the hosted version of this site, GitHub may collect basic server logs (such as IP addresses) as part of their standard security, DDoS protection, and delivery protocols. These logs are managed by GitHub in accordance with the GitHub Privacy Statement.
  • Content Delivery Networks (CDNs): To ensure high performance, we load external libraries like Bootstrap and Prism.js via CDNs. These services may log technical metadata, such as your browser version and device type, to optimize file delivery.

3. Data Security

Because this project is strictly an educational resource and does not collect, store, or transmit sensitive information, the risk of personal data breaches is non-existent. We maintain the source code following security best practices to ensure that your browsing experience remains safe and free of malicious scripts.

4. Open Source Transparency

As an open-source project located within the abstract directory, all source code—including the logic that handles user interface interactions—is fully auditable by the public. You are encouraged to review the index.html and associated scripts to verify our privacy and security claims.

5. Changes to This Policy

We reserve the right to update this Privacy Policy to reflect changes in our project or legal requirements. Any updates will be posted on this page with an updated "Last Updated" date at the top.

6. Contact

If you have any questions or concerns regarding this policy or the technical implementation of this project, please open a formal Issue on the GitHub repository for this project.
